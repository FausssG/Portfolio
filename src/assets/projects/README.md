# Imágenes de Proyectos

Coloca aquí las imágenes de tus proyectos con los siguientes nombres:

- `proyecto-final.png` o `.jpg` - Proyecto Final de Carrera
- `wpc.png` o `.jpg` - Sistema de Gestión WPC
- `sistema-experto.png` o `.jpg` - Sistema Experto React
- `data-science.png` o `.jpg` - Ciencia de Datos
- `simulacion.png` o `.jpg` - Balanceo de Personal
- `datacenter.png` o `.jpg` - Data Center
- `robot.png` o `.jpg` - Robot Seguidor de Líneas
- `veterinaria.png` or `.jpg` - Sistema de Gestión Veterinaria

## Formato recomendado:
- Formato: PNG o JPG
- Resolución: 1200x600px (16:9) o similar
- Tamaño: Menos de 500KB por imagen
- Screenshots de la interfaz o diagramas del proyecto
